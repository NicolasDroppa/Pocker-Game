## Pocker Test ##

5 card draw game,...
This project requires Java 1.8> and Mavan to run.
### Running the program on Linux ###
```bash
  $ cd pocker-test
  $ ./run.sh
```

### Running the program on Windows ###
```bash
  $ cd pocker-test
  $ ./run.bat
```
